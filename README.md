# VRM_AI
Unity3DとChatGPTを連携させることができるソフト。
VRMを読み込める。

# Usage
VRM_AI.zipを解凍しVRM_AI.exeと付属のPythonスクリプトの実行環境を作り、実行して下さい。
詳細はVRM_AI.zip内のreadme.txtに書いてあります。

# unitypakage
